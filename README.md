# Flappy-bird-game

## Description
I was looking through my Codedex account for ideas for new apps to build and I stumbled across a Flappy Bird Game clone tutorial that got me really interested because it was built using Phaser, which is a fairly new topic for me. I haven't done a lot of game building and I really wanted to try something new while still using my roots in HTML and JavaScript and this was the perfect opportunity. So I sat down and followed the tutorial and learned how to create this game from scratch. But I didn't want to leave it at just that basic clone. I wanted it to be more personal and customized. So I decided to make a whole UI inside the game with main screens and end screens and even a settings screen to let users customize their game play. I had a lot of fun searching up how to add overlays to a phaser app and how to connect the HTML front to the Phaser of the game. THis has been a really good project for me to learn more and improve my skills and I hope you enjoy the game!!

This is a simple flappy bird game where you have to click on the up arrow button to keep the bird flying and not hit any of the obstacles. I learned a lot about Game Physics and Collisions while also improving my HTML and JavaScript background by learnig how to make the two seamlessly work with Phaser. 

## Features
- New Game
- Settings
    - Increase Target Score
    - Decrease Target Score
    - Change Theme: City and Cave Themes - click on the "City Theme >" button to toggle
- Play Again - takes directly to new game
- Main Menu - takes to main menu screen
- Buttons to navigate - found on Game Console on screen
    - Main Screen -
        - Top Circle Button - New Game
        - Bottom Circle Button - Settings
    - Game end Screen -
        - Top Circle Button - Play Again
        - Bottom Circle Button - Main Menu
- You can use your mouse to click on the circle buttons as well as the actual buttons on screen too
- Jump to avoid obstacles

## How to Play
- Click on New Game on Screen either using your mouse or the buttons on the console
- Click on your Space bar when you are ready to play (or) click on the square button on the console to start
- As the bird starts falling, click on the upp arrow button on your keyboard to fly up (or) use the square button in the same way to fly up
- Dodge the obstacles
- Reach the Target Score
