# Flappy-bird-game

## Description


## Features
- New Game
- Settings
    - Increase Target Score
    - Decrease Target Score
    - Change Theme: City and Cave Themes - click on the "City Theme >" button to toggle
- Play Again - takes directly to new game
- Main Menu - takes to main menu screen
- Buttons to navigate - found on Game Console on screen
    - Main Screen -
        - Top Circle Button - New Game
        - Bottom Circle Button - Settings
    - Game end Screen -
        - Top Circle Button - Play Again
        - Bottom Circle Button - Main Menu
- You can use your mouse to click on the circle buttons as well as the actual buttons on screen too
- Jump to avoid obstacles

## How to Play
- Click on New Game on Screen either using your mouse or the buttons on the console
- Click on your Space bar when you are ready to play (or) click on the square button on the console to start
- As the bird starts falling, click on the upp arrow button on your keyboard to fly up (or) use the square button in the same way to fly up
- Dodge the obstacles
- Reach the Target Score
